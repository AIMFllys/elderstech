出一、选题背景及意义
“老吾老，以及人之老”。习近平总书记多次强调，“有效应对我国人口老龄化，事关国家发展全局，事关亿万百姓福祉”。这一重要论述深刻阐明了老龄工作在国家发展战略中的核心地位与深远影响。深入开展老龄化健康领域的实践调研，对于探索符合国情、市情的养老服务新模式，完善健康保障体系，助力老龄事业发展具有重要的现实意义。
老龄化挑战升级，催生健康养老新挑战。中国老龄化呈现“规模大、速度快、未富先老”的典型特征。据预测，2035 年老年人口将突破4 亿，占比超 30%，迈入重度老龄化阶段。老龄化与慢性病高发叠加，医疗护理需求激增与传统养老资源短缺的矛盾日益尖锐，探索医养融合与智慧技术的创新协同，已从“前瞻布局”升级为“民生刚需”。国家战略高位布局，着力打造医养融合新生态。党的十八大以来，以习近平同志为核心的党中央高度关心关注老龄化工作，逐步将“医养结合”上升为健康中国战略核心。2016年《“健康中国 2030”规划纲要》突出强调“促 进健康老龄化”“健全医养结合机制”。2025年最新发布的《国务院关于深化养老服务改革发展的意见》提出深化全国智慧健康养老应用示范，推广智能化家居和智慧健康产品。创新应用涌现，为应对老龄化带来新工具、新挑战。智慧养老、远程医疗、智能辅具等创新应用为提升老年生活质量带来了新的可能。然而，在实践中，许多智慧养老产品与数字医疗服务因操作复杂、界面不友好，未能充分考虑老年群体的实际需求与使用习惯，形成了新的“数字鸿沟”。技术本应为老服务，若设计不当，反而可能成为老年人获取现代医疗健康服务的障碍。
本次社会实践项目正是基于这样的时代背景与现实关切，以习近平新时代中国特色社会主义思想为指导，锚定“积极应对人口老龄化国家战略”，聚焦老年群体这一核心对象，创新提出探索智慧工具给老年群体带来的双重影响与解决方案，同时创新提出了“托起幸福夕阳红”智慧医养专项计划，综合运用文献研究、问卷调查、实地走访、深度访谈等调研方式，按照调查研究-认知升华-实践创造”的知行合一路径深刻理解老年人在健康管理等方面的真实需求；系统评估其现有实际成效与存在问题；探索医学知识与人工智能技术相结合的服务创新可能。最终找到真正适老化的、具有中国特色的立体化智慧医养服务新模式。

二、实践主要内容
1. 返乡摸底调研，洞察基层现状
组织队员返回家乡，采取“集中指导、分散调研”模式，深入养老机构、医院老年科室及老年人家庭。通过问卷调查与深度访谈，系统调研当前医养服务的供给模式、资源配备及利用情况，重点收集老年人对智慧产品与在线医疗服务的真实使用体验、主要困难及改进期望。
2. 开展案例研究，明晰关键问题
围绕“智慧工具是桥梁还是壁垒”的核心问题，对调研中发现的典型使用场景进行深度案例剖析。追踪老年人接触、学习、使用或放弃使用某项智能技术的全过程，深入分析影响其技术接受度的主观与客观因素，形成有深度的案例报告，为服务改进提供依据。
3. 推动融合创新，开发适老成果
启动“智护银龄”辅助工具开发项目。核心是设计并开发一款面向老年人的“智能用药与健康导航”轻应用原型，聚焦用药提醒与健康信息简易查询功能。同时，配套设计“适老化图标包”、“药物分类色卡”等实体辅助工具，并开发有声书等互动式健康宣教材料，以多形态创新成果提升服务可及性与友好度。
4. 总结调研发现，贡献青年方案
在全面调研与案例分析基础上，系统梳理智慧医养服务在基层落地面临的现实矛盾与根本原因。最终撰写实践报告，旨在超越单纯的情况分析，从老年人真实需求出发，提出具有建设性、可操作性的服务优化与产品改进方案，为构建中国特色立体化智慧医养服务体系建言献策。

三、实践具体方案
时间：2026年1月25日至2月22日（共约5周）。
地点：覆盖团队成员所在的多个省份，包括湖北武汉、黑龙江佳木斯、安徽合肥、山东烟台、河北张家口康保县、内蒙古呼和浩特赛罕区、贵州黔南州等具有地域代表性的市县乡。
方法：综合运用文献研究、问卷调查、实地走访、深度访谈等方法。
进度安排：
第一阶段（1.25-1.27）准备启动：召开线上培训会，统一调研标准与方法。
第二阶段（1.27-2.17）分散调研与同步开发：成员返乡开展为期三周的实地调研，每周提交日志；开发组同步推进应用原型与产品设计。
第三阶段（2.17-2.20）集中分析：进行数据汇总与案例分析，召开线上研讨会，分工撰写报告。
第四阶段（2.21-2.22）成果总结：完成报告统稿、润色与排版，完善应用原型及相关成果设计。

四、已有实践基础
1. 组织与保障有力：本项目为基础医学院重点立项的实践队伍，已纳入院系“实践育人”重点工程及年度经费预算，管理体系成熟，保障充分。上一届奠定良好基础，实践走访合肥、如皋、武汉三地17家机构，完成百余问卷，覆盖7万老年人，提出12项建议，形成4份专项建议函，服务获5家媒体报道，形成共计3万字的社会实践调研报告。
2. 数据与资源扎实：团队已预先收集整理逾20万字的智慧医养相关文献资料。同时，得益于学院过往实践（如“齐鲁青未了”乡村医养实践队）在山东多地的数据积累，为本项目提供了宝贵的对比参考。目前，多数实践地点的对接联络工作已完成，确保了调研的可行性。
3. 团队构成专业：团队由学院党委副书记、团委书记及辅导员共同指导，提供学术与组织保障。实践队伍由来自多个专业的10名本科生构成，兼具医学背景与计算机技术能力，且成员均具备丰富的学生工作与社会实践经验，曾多次获得校级及以上社会实践荣誉，具备强大的执行力与创新潜力。
4. 成果导向明确：团队注重创新与成果转化。计算机专业成员在算法与软件开发方面具备优势，医学生则深刻理解老年健康需求，两者的跨学科融合将有力推动“智能用药与健康导航”应用原型等创新成果的产出，确保项目在智慧医养领域实现实质性探索与突破。
