### 二、 背景 (Backgrounds)

**1. 核心定义**
*   **颜色体系 (Tailwind Config)**：
    *   `hust`: `#b8860b` (金棕色，主题色)
    *   `ink`: `#5a4b3c` (墨色，主要文字色)
    *   `paper`: `#f8f5f0` (纸张色，主要背景色)
*   **视差背景 (`bg-parallax`)**：
    *   **普通模式**：浅米色背景 (`#f8f5f0`) + 0.9 透明度遮罩 + 固定定位的图片 (`picsum.photos/id/307`)。
    *   **深色模式**：深蓝色背景 (`#111827`) + 0.94 高透明度黑遮罩 + 同款图片。
*   **动态效果**：包含一个全屏固定的 `canvas` (`#floatCanvas`)，用于渲染漂浮的文字（如“阅读”、“思考”等）。

**2. 相关代码**
```javascript
// Tailwind 颜色配置
colors: {
  hust: { DEFAULT: '#b8860b', dark: '#8a6608', light: '#e6c875' },
  ink: { DEFAULT: '#5a4b3c', dark: '#e8e6e3' },
  paper: { DEFAULT: '#f8f5f0', dark: '#1a1a1a' }
}
```

```css
/* 视差背景 CSS */
body.bg-parallax {
  background-color: #f8f5f0 !important; 
  background-image: 
    linear-gradient(rgba(248, 245, 240, 0.9), rgba(248, 245, 240, 0.9)),
    url("https://picsum.photos/id/307/1920/1080") !important;
  background-attachment: fixed !important; /* 背景固定，产生视差感 */
  background-size: cover !important;
}

/* 深色模式背景切换 */
html.dark body.bg-parallax {
  background-color: #111827 !important;
  background-image: 
    linear-gradient(rgba(20, 20, 20, 0.94), rgba(20, 20, 20, 0.94)),
    url("https://picsum.photos/id/307/1920/1080") !important;
}

/* 动态漂浮层 */
#floatCanvas { 
  position: fixed; 
  inset: 0; 
  z-index: -1; 
  pointer-events: none; 
}
```
